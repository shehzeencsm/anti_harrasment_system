# Anti-Harassment System
A web-based platform to report and manage harassment cases with police and rescue support.
#project overview
This project helps users report harassment incidents safely.
It includes separate panels for users, police, and girls rescue teams
to ensure quick action and victim protection.
## Features
- User panel for submitting reports
- Anonymous complaint system
- Police dashboard for case management
- Girls rescue emergency support
- Case status tracking
- Secure data handling
